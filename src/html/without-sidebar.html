Without sidebar

You can remove the sidebar by default just adding the class without-sidebar into the main wrapper element (div#wrapper) and removing the sidebar main element (div.sidebar).

<!DOCTYPE html>
<html lang="en">
<head>
  ...
</head>
<body class="with-overlay-scrollbars">
  <div id="wrapper" class="without-sidebar">
    <nav class="navbar navbar-default navbar-fixed-top" role="navigation">
      ...
    </nav>

    <div id="page-wrapper">
      ...
    </div>
    ...
  </div>
</body>
</html>
