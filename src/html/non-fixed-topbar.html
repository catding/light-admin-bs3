Non-fixed topbar

You can set the non-fixed topbar by default just replacing the class navbar-fixed-top to navbar-static-top and removing class fixed-topbar from the main wrapper element (div#wrapper).

<!DOCTYPE html>
<html lang="en">
<head>
  ...
</head>
<body class="with-overlay-scrollbars">
  <div id="wrapper" class="">
    <nav id="topbar" class="navbar navbar-default navbar-static-top">
      ...
    </nav>
    ...
  </div>
</body>
</html>
