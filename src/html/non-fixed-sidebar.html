Non-fixed sidebar

You can set the non-fixed sidebar by default just removing the class sidebar-fixed from the main wrapper element (div#wrapper) and removing class overlay-scrollbar from div.sidebar.

<!DOCTYPE html>
<html lang="en">
<head>
  ...
</head>
<body class="with-overlay-scrollbars">
  <div id="wrapper" class="">
    ...
    <div id="sidebar" class="sidebar">
      ...
    </div>
  </div>
</body>
</html>
